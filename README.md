## Cargo

`cargo new blog` - Create a new project

`cargo check` - Compile and find syntax, perf and other problems

`cargo build` - Compile

`cargo update` - Update dependencies listed in Cargo.lock

## Update Rust

```
rustup --version
rustup check
rustup update
```
